Educational institutions often struggle to provide personalized mentoring at scale. Traditional mentoring systems rely heavily on manual judgment, which limits scalability and early risk detection.

This project proposes a Dedicated Mentoring System powered by a hybrid AI framework combining rule-based intelligence and machine learning. The system processes student academic, wellness, productivity, and career-related data to generate performance scores, identify hidden student clusters using K-Means clustering, and assign mentors based on expertise and availability.

Technologies used include Python, pandas, scikit-learn, and data visualization tools. The system successfully demonstrates risk detection, student segmentation, mentor matching, and intervention recommendation, forming a scalable foundation for AI-driven student support.
